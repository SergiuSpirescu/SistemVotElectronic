# SistemVotElectronic
 Proiect pentru un Sistem de Vot Electronic

Scopul acestui Proiect este experimentarea cu mai multe limbaje de programare si tehnologii pentru a crea un Sistem Electronic de Vot care sa fie:
  - sigur
  - privat
  - intuitiv
  - universal
  - robust
